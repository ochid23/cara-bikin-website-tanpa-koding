# Design System

## Overview

Design terinspirasi dari website Payop dengan gaya:

- Modern SaaS Landing Page
- Clean & Minimal
- Rounded UI
- Gradient Blue
- Soft Shadow
- Premium Feeling
- White Space yang luas
- Responsive First

# UI Principles

- Banyak white space
- Gunakan card besar
- Rounded corner
- Soft shadow
- CTA selalu kontras
- Dominasi warna putih dan biru
- Ilustrasi modern
- Motion ringan
- Tipografi tebal untuk headline
- Visual premium seperti Stripe, Payop, Vercel, dan Linear
